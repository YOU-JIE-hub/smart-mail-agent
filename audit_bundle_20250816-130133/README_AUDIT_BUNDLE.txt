This bundle is for final review:
- reports/tree-*.txt               : repository tree
- reports/tests-collect-*.txt      : pytest collection
- reports/coverage-offline.xml     : coverage xml (offline suite)
- reports/html-offline/            : coverage html report
- assets/badges/coverage.svg       : local coverage badge
- config/                          : README, pyproject, requirements, pre-commit, gitignore/gitattributes, pytest.ini, workflows
- src_lowcov/                      : source files with line-rate < 60% (from coverage)
- LOW_COVERAGE_FILES.txt           : list of low-coverage files
